# Usage/Installion
 
Ensure your shell is a zsh & you have make installed

```bash
git clone git@github.com:ashokrajar/dotfiles.git ~/.dotfiles
cd ~/.dotfiles

make install
```
 
# UnInstall
 
```bash
make clean
```
